# Lavender Sweet

Lavender Sweet is an elegant e-commerce website built with **HTML, CSS, PHP, and MySQL**.

## Project Structure
- PHP pages for all website functions
- CSS files for styling
- Images in `images/` folder
- Database file `lavender.sql`

## How to Run
1. Place project folder in `htdocs` (XAMPP)
2. Start Apache & MySQL in XAMPP
3. Import `lavender.sql` in phpMyAdmin
4. Open browser at http://localhost/lavender-sweet/
